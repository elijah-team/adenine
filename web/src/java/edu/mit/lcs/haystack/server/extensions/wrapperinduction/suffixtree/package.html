ahogue:An implementation of the Suffix Tree data structure for manipulating strings, with an extension to hierarchical structures.

Owner: ahogue
Status: In progress
Date: 3/2/04

This package contains an implementation of the Suffix Tree data structure, originally described by P. Weiner, Linear pattern matching algorithms, in Proc. 14th IEEE Symposium on Switching and Automata Theory, pg 1-11, 1973.

In addition, an extension to hierarchical structures, the Suffix Multitree, is included, as described here.

Both structures contain functions for locating repetitive structures in their respective data types (strings or trees). The Suffix Tree implementation is complete and functional. The Suffix Multitree implementation is also functional, although more testing is required.
