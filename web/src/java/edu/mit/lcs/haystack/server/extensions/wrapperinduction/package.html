ahogue: The base wrapper induction classes and interfaces.

Owner: ahogue
Status: In progress
Date: 3/2/04

This package contains the basic interfaces and classes that make up the wrapper induction implementation. Wrapper induction is a method for deriving patterns on the Web based on user input. For more information, see my thesis proposal.

Classes in this package depend on some of the core Haystack RDF functionality, especially @see edu.mit.lcs.haystack.rdf.

Currently, there should be no dependencies of other packages on the classes in this package.
