ahogue: Test classes for the wrapper induction package.

Owner: ahogue
Status: In progress
Date: 3/2/04

This is a test package for the classes and interfaces in @see edu.mit.lcs.haystack.wrapperinduction. Currently it contains a wrapper induction interface using Internet Explorer, along with some stand-alone applications for testing.

Classes in this package depend on some of the core Haystack RDF functionality, especially @see edu.mit.lcs.haystack.rdf, as well as some ozone parts (@see edu.mit.lcs.haystack.ozone).

Currently, there should be no dependencies of other packages on the classes in this package.

Further references are listed below.
@see edu.mit.lcs.haystack.server.extensions.wrapperinduction
