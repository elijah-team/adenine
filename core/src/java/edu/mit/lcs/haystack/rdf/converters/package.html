matsakis:Converters between RDF and other file formats (VCARD, etc.)

Owner: matsakis
Status: on hold
Date: 3/2/04

This is a skeleton of a framework for converting between different file formats, text or otherwise, and RDF. The basic idea is that for each format we need to define a 'parser' and a 'generator'. The hope is that we can maintain a mapping between MIME types and this code, so that when the user wants to import/export data the appropriate converter can be found dynamically.

This package should depend on no other part of Haystack besides the RDF framework and java core libraries. Further references are listed below. @see edu.mit.lcs.haystack.rdf
