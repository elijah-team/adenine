Owner: haystack
Status: core
Date: 3/1/04

This package consists of classes representing the core of Haystack:

  • Haystack: the Haystack platform singleton
  • Constants: Haystack-specific URI and namespace constants
  • HaystackException: baseclass of all Haystack system exceptions
  • ExceptionsFilterConsoleAppender: log4j appender that filters out exception, used at command prompt output

There are utility classes that shouldn't be in this package:

  • MapEntry
  • MapValueComparator
  • Pair
  • ReaderInputStream: for converting a reader into an input stream

There are classes that I don't know about:

  • GenerateUserID
  • IExternalClassLoader
  • SynchronizationManager
